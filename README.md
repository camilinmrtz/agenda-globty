# Agenda Globty — Tech Recruiter (Package)

Esto es un paquete listo para desplegar con Vite + React.

## Pasos rápidos (local)

1. Copia `.env.example` a `.env` y completa tus credenciales.
2. Instala dependencias:
   npm install
3. Levanta en desarrollo:
   npm run dev
4. Abre http://localhost:5173

## Deploy en Vercel

1. Subí el repo a GitHub.
2. Importá el repo en Vercel.
3. Configurá las env vars en Vercel (si no usas window.__ENV).
4. Deploy y listo.
